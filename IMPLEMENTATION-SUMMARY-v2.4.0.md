# PowerSCAP v2.4.0 - Auto-Download Feature

PowerSCAP v2.4.0 adds automatic vulnerability definition download from NVD, MSRC, and OVAL archives.

## Key Features
- Download from NVD (National Vulnerability Database)
- Download from MSRC (Microsoft Security Response Center)  
- Download from archived OVAL repositories
- Smart caching by date
- NVD API key support

## Implementation Date
February 2, 2026

See AUTO-DOWNLOAD-GUIDE.md for complete documentation.
